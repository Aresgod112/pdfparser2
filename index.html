About This Application

The PDF Parser application extracts key information from contract PDFs:

  • Effective Date
  • Start Date
  • Initial Term
  • Further Term

It supports both text-based PDFs and scanned documents (using OCR).

Features

  • Upload Multiple PDFs: Process one or multiple contract PDFs at once
  • Smart Text Extraction: Uses standard PDF libraries for text-based PDFs and OCR for scanned documents
  • Enhanced Results Display: Shows extracted data in a well-formatted tabular format with filtering options
  • Export Options: Download results as CSV or Excel files

How to Run This Application

To use this application, you need to run it locally:

  1. Clone the repository from GitHub:
    git clone https://github.com/Aresgod112/pdfparser.git
    cd pdfparser
  2. Install the required dependencies:
    pip install -r requirements.txt
  3. Run the application:
    streamlit run app.py
